Champions Speed Explorer

How to use
  1. Type a Pokémon name in Add Pokemon and click Add.
  2. Pick an entry from the Selected Pokemon dropdown to view its card.
  3. Adjust Nature, SP, and Stage — the chart updates instantly.
  4. Click any chart badge to jump to that entry's card.
  5. Use Export / Import to save and reload configs.
